# gostub

Simple http stub server by golang

![gostub](./gostub.png)

## Usage

```
Usage of ./gostub:
  -o string
    	output path (e.g. 'tests' -> ./tests)
  -p string
    	port number (default "8181")
```

## Requirement

Only Go language

## Installation

```sh
go get https://github.com/moaible/gostub

```
